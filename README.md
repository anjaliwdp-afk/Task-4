# Blog CRUD Application

A full-featured CRUD (Create, Read, Update, Delete) web application with user authentication, developed during my internship at ApexPlanet Software Pvt Ltd. Users can register, log in, and manage their own blog posts securely.

**Tech Stack:** PHP, MySQL, HTML5, CSS3, XAMPP

**Key Features:**
- Secure user authentication (password hashing)
- Session management
- Create, read, update, delete posts
- User-specific content
- Responsive design

This project demonstrates backend development, database integration, and practical implementation of CRUD operations.
